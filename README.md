# Computer Graphics - M.Sc. Computer Science, University of Bologna

This repository contains various materials related to the Computer Graphics course for the M.Sc. Computer Science program at the University of Bologna, taught by Professor [Giulio Casciola](https://www.dm.unibo.it/~casciola/) for the academic year 2025-2026.

You can find almost all the materials on the related [GitHub Website](https://vaiton.github.io/Unibo-CG-Spring-2026/).
